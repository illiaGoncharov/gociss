# Руководство по работе с ACF

Advanced Custom Fields (ACF) - это плагин для WordPress, который позволяет создавать кастомные поля для записей, страниц и других типов контента.

## Что такое ACF?

ACF позволяет добавлять дополнительные поля к страницам и записям WordPress без программирования. Например, можно добавить поле "Заголовок Hero секции" или "Список услуг".

## Где находятся ACF поля?

ACF поля регистрируются в файле `inc/acf-fields.php`. После активации плагина ACF, эти поля автоматически появятся в админке WordPress.

## Как редактировать контент через ACF?

### На главной странице

1. Зайди в админку WordPress
2. Перейди в **Страницы → Все страницы**
3. Найди главную страницу (та, что использует шаблон "Главная страница")
4. Открой страницу для редактирования
5. Прокрути вниз - ты увидишь блоки с ACF полями:
   - **Hero секция** - заголовок, описание, кнопки, статистика
   - **Секция Услуги** - список услуг
   - **Секция Преимущества** - список преимуществ
   - **Секция FAQ** - вопросы и ответы

### Редактирование Hero секции

В блоке "Hero секция" ты найдёшь:

- **Метка** - маленькая синяя метка над заголовком
- **Заголовок** - главный заголовок страницы
- **Описание** - текст под заголовком
- **Изображение** - картинка справа от текста
- **Основная кнопка** - группа полей с текстом и ссылкой
- **Вторичная кнопка** - вторая кнопка
- **Статистика** - повторяющееся поле (репитер) с числами и подписями

### Редактирование услуг

В блоке "Секция Услуги":

- **Заголовок** - заголовок секции (например, "Ключевые направления")
- **Подзаголовок** - описание под заголовком
- **Услуги** - репитер, где каждая услуга содержит:
  - Иконку (изображение)
  - Заголовок
  - Описание
  - Ссылку

Чтобы добавить новую услугу:
1. Нажми кнопку "Добавить строку" в блоке "Услуги"
2. Заполни поля для новой услуги
3. Сохрани страницу

### Редактирование преимуществ

Аналогично услугам, но без ссылок. Каждое преимущество содержит:
- Иконку
- Текст описания

### Редактирование FAQ

В блоке "Секция FAQ":

- **Заголовок** - заголовок секции
- **Подзаголовок** - описание
- **Вопросы** - репитер с:
  - Вопросом (текст)
  - Ответом (текстовый редактор, можно использовать форматирование)

## Работа с экспертами

Эксперты - это отдельный тип записей. Чтобы добавить эксперта:

1. Перейди в **Эксперты → Добавить нового**
2. Заполни заголовок (ФИО эксперта)
3. Загрузи фото эксперта (миниатюра записи)
4. В блоке "Данные эксперта" заполни:
   - **Должность** - например, "Генеральный директор"
   - **Опыт** - описание опыта, например "12 лет опыта, более 500 проектов"
5. Сохрани запись

Эксперты автоматически появятся в секции "Наши ведущие эксперты" на главной странице.

## Работа с новостями

Новости - это обычные записи WordPress:

1. Перейди в **Записи → Добавить новую**
2. Заполни заголовок и текст новости
3. Добавь изображение (миниатюра записи)
4. Опубликуй запись

Последние 4 новости автоматически появятся в секции "Новости и статьи" на главной странице.

## Типы полей ACF

В теме используются следующие типы полей:

- **Text** - обычное текстовое поле
- **Textarea** - многострочное текстовое поле
- **Image** - загрузка изображения
- **URL** - поле для ссылки
- **Group** - группа полей (например, для кнопки с текстом и ссылкой)
- **Repeater** - повторяющееся поле (список элементов)

## Советы

1. **Всегда заполняй обязательные поля** - они помечены звёздочкой (*)
2. **Используй качественные изображения** - они должны быть оптимизированы для веба
3. **Проверяй ссылки** - особенно в кнопках и услугах
4. **Сохраняй изменения** - не забывай нажимать "Обновить" после редактирования

## Экспорт/импорт полей

Если нужно перенести ACF поля на другой сайт:

1. Перейди в **Поля → Инструменты**
2. Выбери группу полей
3. Скопируй JSON код
4. На новом сайте вставь код в **Поля → Инструменты → Импорт**
