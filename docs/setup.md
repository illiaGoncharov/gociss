# Начальная настройка проекта

Это руководство поможет настроить проект с нуля.

## Требования

- PHP 8.0 или выше
- WordPress 6.0 или выше
- MySQL 5.7 или выше
- Node.js 18+ (для разработки)
- Git

## Установка WordPress

### Вариант 1: Docker (Рекомендуется для разработки)

Самый простой способ для локальной разработки:

```bash
# Запустить окружение
docker-compose up -d

# Открыть сайт
open http://localhost:8080
```

Подробная инструкция в [документации по Docker](docker-setup.md).

### Вариант 2: Традиционная установка

1. Скачай WordPress с [официального сайта](https://wordpress.org/download/)
2. Распакуй архив в папку проекта
3. Создай базу данных MySQL для WordPress
4. Запусти установку WordPress через браузер

## Установка темы

1. Скопируй папку `wp-content/themes/gociss-theme` в папку `wp-content/themes/` твоего WordPress
2. Активируй тему в админке WordPress: **Внешний вид → Темы**

## Установка плагинов

### Обязательные плагины

1. **Advanced Custom Fields (ACF)**
   - Установи через админку: **Плагины → Добавить новый**
   - Найди "Advanced Custom Fields" и установи
   - Активируй плагин

### Рекомендуемые плагины

- **Polylang** (для мультиязычности) - см. [документацию по мультиязычности](multilingual.md)
- **Contact Form 7** (если нужны дополнительные формы)
- **Yoast SEO** (для SEO оптимизации)
- **WP Super Cache** (для кеширования)

## Настройка ACF полей

После активации ACF плагина:

1. Зайди в админку WordPress
2. Перейди в **Поля → Группы полей**
3. Ты увидишь группы полей, которые уже зарегистрированы в коде
4. Если нужно изменить поля, редактируй их через интерфейс ACF
5. Экспортируй изменения в код (опционально)

## Настройка главной страницы

1. Создай новую страницу в WordPress: **Страницы → Добавить новую**
2. Назови её "Главная" или "Home"
3. В атрибутах страницы выбери шаблон "Главная страница (page-front.php)"
4. Перейди в **Настройки → Чтение**
5. Выбери "Статическая страница" и укажи созданную страницу как главную

## Настройка меню

1. Перейди в **Внешний вид → Меню**
2. Создай новое меню или используй существующее
3. Добавь пункты меню
4. В разделе "Расположение меню" выбери "Основное меню"
5. Сохрани изменения

## Настройка для разработки

### Установка зависимостей Node.js

```bash
npm install
```

### Запуск линтеров

```bash
# Проверка JavaScript
npm run lint:js

# Проверка CSS
npm run lint:css

# Проверка всего
npm run lint
```

## Настройка GitHub Actions

Для работы автоматического деплоя нужно настроить секреты в GitHub:

1. Перейди в настройки репозитория: **Settings → Secrets and variables → Actions**
2. Добавь следующие секреты:
   - `BEGET_FTP_HOST` - адрес FTP сервера Бегет
   - `BEGET_FTP_USER` - логин FTP
   - `BEGET_FTP_PASSWORD` - пароль FTP

## Первые шаги

1. Заполни ACF поля на главной странице
2. Добавь несколько экспертов через **Эксперты → Добавить нового**
3. Создай несколько новостей через **Записи → Добавить новую**
4. Проверь работу формы обратной связи
