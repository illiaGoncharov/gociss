# Быстрый старт

Краткая инструкция для начала работы с темой.

## Первый запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/твой-username/gociss-theme.git
cd gociss-theme
```

### 2. Запусти Docker

```bash
make up
```

Подожди 15-20 секунд, пока контейнеры запустятся.

### 3. Установи WordPress

```bash
make install
```

Или установи вручную через браузер:
- Открой http://localhost:8080
- Заполни форму установки

### 4. Активируй тему

1. Зайди в админку: http://localhost:8080/wp-admin
2. Перейди в **Внешний вид → Темы**
3. Активируй тему "ГоЦИСС"

### 5. Установи плагины

1. Перейди в **Плагины → Добавить новый**
2. Установи и активируй:
   - **Advanced Custom Fields (ACF)** - обязательно
   - **Polylang** - для мультиязычности (если нужно)

## Полезные команды

```bash
# Запустить окружение
make up

# Остановить
make down

# Просмотр логов
make logs

# Войти в контейнер
make shell

# Создать бэкап БД
make db-backup

# Показать все команды
make help
```
