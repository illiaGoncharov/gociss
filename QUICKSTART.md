# Быстрый старт

## После установки WordPress

### 1. Активируй тему

1. Зайди в админку: http://localhost:8080/wp-admin
2. Перейди в **Внешний вид → Темы**
3. Активируй тему "ГоЦИСС"

### 2. Создай главную страницу

**Вариант А: Автоматически (при активации темы)**

Тема автоматически создаст страницу "Главная" при активации.

**Вариант Б: Вручную**

1. Перейди в **Страницы → Добавить новую**
2. Название: **Главная**
3. В атрибутах страницы (справа) выбери шаблон: **Главная страница (page-front.php)**
4. Нажми **Опубликовать**

### 3. Настрой главную страницу

1. Перейди в **Настройки → Чтение**
2. Выбери **Статическую страницу**
3. В выпадающем списке "Домашняя страница" выбери **Главная**
4. Нажми **Сохранить изменения**

### 4. Установи плагины

1. Перейди в **Плагины → Добавить новый**
2. Установи и активируй:
   - **Advanced Custom Fields (ACF)** - обязательно для работы темы
   - **Polylang** - для мультиязычности (если нужно)

### 5. Создай меню

1. Перейди в **Внешний вид → Меню**
2. Создай новое меню или используй существующее
3. Добавь пункты: О компании, Аккредитация, Реестры, Блог, Контакты
4. В разделе "Расположение меню" выбери **Основное меню**
5. Нажми **Сохранить меню**

### 6. Заполни контент

1. Открой страницу "Главная" для редактирования
2. Прокрути вниз - ты увидишь блоки ACF полей:
   - **Hero секция** - заголовок, описание, кнопки, статистика
   - **Секция Услуги** - список услуг
   - **Секция Преимущества** - список преимуществ
   - **Секция FAQ** - вопросы и ответы
3. Заполни поля и сохрани

### 7. Проверь сайт

Открой http://localhost:8080 - должна отображаться главная страница с дизайном темы.

## Полезные команды Docker

```bash
# Запустить окружение
make up

# Остановить
make down

# Просмотр логов
make logs

# Войти в контейнер
make shell
```
